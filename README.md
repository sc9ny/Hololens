This project is sponsored by Advanced Research Computing Lab, Visionarium at Virginia Tech.
The goal for this project is to visualize wifi signal virtually on Microsoft Hololens.
The project includes usage of Unity for GUI and graphic development(front-end) and C# in Visual Studio for back-end development.
